# CodeAlpha_BasicChatBot
A basic if-else chatbot using python.
Project 2: Rule-Based Chatbot
Description
A conversational program that responds to text inputs using predefined patterns. It handles greetings, questions, farewells, jokes, and help requests with multiple response variations for natural conversation. The bot runs continuously until exit, processes case-insensitive input, and provides friendly fallback messages for unrecognized phrases, demonstrating core programming concepts.
Features

Pattern-based response matching
Multiple response variations for natural conversation
Case-insensitive input processing
Continuous conversation loop
Help system
Graceful handling of unknown inputs
Exit commands

How to Run
bashpython chatbot.py
Supported Commands

Greetings: hello, hi, hey
Status: how are you, how's it going
Identity: what's your name, who are you
Help: help, what can you do
Thanks: thank you, thanks
Jokes: tell me a joke, joke
Age: how old are you
Exit: bye, goodbye, quit, exit

Key Concepts Demonstrated

Functions for code organization
If-elif-else conditional statements
While loops for continuous operation
Input/output handling
String methods (.lower(), .strip())
Lists and random selection
Default fallback responses
